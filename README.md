# Flask Backend Authentication API

This project is a backend authentication system built using Flask. It includes user registration and login functionality with secure password hashing and token-based authentication.

## Features
- User registration with password hashing (Werkzeug)
- Secure login system
- JWT-based authentication
- Protected routes
- SQLite database integration
- Modular project structure using Blueprints

## Technologies Used
- Python
- Flask
- SQLite
- PyJWT
- Werkzeug

## What I Learned
- How backend authentication systems work
- Importance of password hashing
- Difference between session-based and token-based authentication
- Structuring Flask projects for scalability

## How to Run
1. Install dependencies: `pip install -r requirements.txt`
2. Run: `python app.py`
3. Test APIs using Postman
